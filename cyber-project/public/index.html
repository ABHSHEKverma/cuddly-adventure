Security Gateway

This entrance is protected by a rate‑limiting controller.
Only a limited number of users may access the site at the same time.

How it works

  1. You click Request Access.
  2. The server checks the number of active users.
  3. If a slot is available → you enter the site.
  4. If all slots are full → you see a "Server Busy" page.
  5. Slots are released automatically after 60 seconds.
